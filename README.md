## 🍭 Characteristics
- Input city and country and receive live weather details 


## 📦 Installation & Setup
```bash
$ git clone https://github.com/emanuallan/weather-app.git
$ cd weather-app
$ yarn install
$ yarn start
```

- This should run the web applet and allow you start inputting values.
